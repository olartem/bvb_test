# README

* Ruby version: 3.0.2
* Ruby on Rails version: 6.1.4
* DB: PostgreSQL 10

## Installation
1. Configure database.yml file
2.
```bash
bundle install
```
3.
```bash
rails db:setup
```
4.
```bash
rails webpacker:install
```
5.
```bash
rails s
```
